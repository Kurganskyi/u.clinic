# Миграции базы данных

Используется Alembic для управления миграциями схемы БД.

## Создание миграции

```bash
alembic revision --autogenerate -m "Описание изменений"
```

## Применение миграций

```bash
# Применить все миграции
alembic upgrade head

# Откатить последнюю миграцию
alembic downgrade -1

# Проверить текущую версию
alembic current
```

## Примечание

Первоначальная структура БД создаётся автоматически при первом запуске бота через `init_db()` в `bot/database.py`. 

Alembic миграции нужны для последующих изменений схемы в production окружении.
