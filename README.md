# U.Clinic

Клиника экспертной косметологии Uclinic — система автоматизации и улучшения коммуникаций с клиентами.

## Описание проекта

Проект включает в себя:
- Анализ сайта и рекомендации по улучшению
- Telegram-бот для автоматизации записи и консультаций
- Система уведомлений и напоминаний

## Основные направления улучшений

### Сайт Uclinic
- Упрощение процесса записи (единый модальный CTA)
- Оптимизация производительности (WebP, lazy-loading, критический CSS)
- Улучшение SEO (микроразметка, шаблоны страниц услуг)
- Сквозная атрибуция звонков и сообщений

### Telegram-бот
- Единый поток записи со слотами расписания
- Автоматизация напоминаний (t-24ч, t-3ч)
- Квиз-подбор процедур
- Интеграция с CRM и аналитикой

## Принципы разработки

- Все конфигурационные данные и credentials загружаются из переменных окружения (.env)
- Денежные суммы в интеграциях представлены в копейках
- Секреты не должны начинаться с префикса "GITHUB_"
- Тесты и документация создаются только по явному запросу
